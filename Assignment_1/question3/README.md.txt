Στην main συνάρτηση δημιουργώ μια λίστα με διάφορα νούμερα καλώντας την insert_at_start για να εισάγει τους καινούριους 
κόμβους.Έπειτα καλώ την PrintList για να μου εκτυπώσει την λίσα και στη συνέχεια την αναδρομκή συνάρηση όπου θα μου 
επιστρέψει το μέγιστο στοιχείο της λίστας. Η αναδρομκή συνάρηση που καλώ για να μου επιστρέψει το μέγιστο στοιχείο της 
λίστας είναι η rec_MAX.Η rec_MAX ξεκινάει με μια μεταβλητή x που της έχω εισάγει σαν όρισμα με τη τμή 0 αυτό που κάνει 
είναι να ελέχει όλους τους κόμβους με το x.Αν το x δεν είναι μεγαλύτερο από τον τρέχοντα κόμβο τοτε το x θα πάρει την τιμή 
του κόμβου και θα καλέσει ξανά την rec_MAX.Αντίστοιχα αν το x είναι μεγαλύτερο δεν αλλάζει η τιμή του και θα καλέσει ξανά 
την rec_MAX. Αυτό θα γίνεται μέχρι να φτάσουμε στο τέλος της λίστας και θα επιστρέψουμε το τρέχων x.
